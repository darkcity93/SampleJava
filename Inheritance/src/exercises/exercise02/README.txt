[문제2] 클래스 타입 변환에 대한 설명 중 틀린 것은?
정답 : 2번
해설 : 명시적으로 타입변환을 할 수 있다. 실행오류가 발생한다.

1. 자식 객체는 부모 타입으로 자동 타입 변환 된다.
2. 부모 객체는 항상 자식 타입으로 강제 타입 변환 된다.
	- 부모 객체를 자식 객체로 ㅌ캐스팅을 하면 실행 오류가 발생한다.
	- 예외발생(java.lang.ClassCastException)
3. 자동 타입 변환을 이용해서 필드와 매개 변수의 다형성을 구현한다.
4. 강제 타입 변환 전에 instsanceof연산자로 변환 가능한지 검사하는 것이 좋다.
